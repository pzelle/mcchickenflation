About McChickenflation

McChickenflation tracks the standalone price of the McChicken over time so we can visualize what inflation actually feels like. Each year captures the minimum and maximum price points we can find, with sources and notes so you can explore the underlying context. Note: This data is far from rigorous, but I don't see many people out there aggregating McChicken prices. So count yourself McLucky.
